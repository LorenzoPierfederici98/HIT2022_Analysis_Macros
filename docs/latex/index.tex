Welcome to the documentation for the HIT 2022 Analysis project.\hypertarget{index_autotoc_md0}{}\doxysubsection{\texorpdfstring{Overview}{Overview}}\label{index_autotoc_md0}
This project contains a set of macros used to analyze Calorimeter and TOF-\/wall data from the HIT2022 campaign, focus of my MS thesis work. These macros take as input the ROOT output files produced by the macros which analyze the runs decoded by the SHOE software, producing the histograms of interest. The ROOT output files of the runs characterized by the same beam energy value are merged into a single file, which is then used as input for these macros.\hypertarget{index_autotoc_md1}{}\doxysubsection{\texorpdfstring{Structure}{Structure}}\label{index_autotoc_md1}
