\doxysubsection{Detailed Description}
Header file for plotting charge distribution histograms from ROOT files. 

Macro that plots the charge given by the fit performed in Analyze\+Peak\+Crystal.\+cc in all the available crystals vs the beam energy. The fit results are stored in objects, named like {\ttfamily Fit\+\_\+result\+\_\+\+Charge\+\_\+\+Calo\+\_\+crystal\+Id\+\_\+1} inside the e.\+g. {\ttfamily Calo/\+Ana\+FOOT\+\_\+\+Calo\+\_\+\+Decoded\+\_\+\+HIT2022\+\_\+140\+Me\+V\+\_\+\+Fit.\+root} file. To be run with `root -\/l -\/b -\/q \textquotesingle{}Calo\+Peak\+Energy\+Display.\+cc()'\`{} it loops on every crystal\+ID and extracts the fit values only on the available IDs.
